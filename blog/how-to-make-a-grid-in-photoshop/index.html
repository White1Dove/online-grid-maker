How to Make a Grid in Photoshop or with an Online Grid Maker

Photoshop can show a grid while you edit. An online grid maker is better when you want a simple gridded image you can save, print, or use as a drawing reference.

There are two common reasons people search for how to make a grid in Photoshop. Some want a layout grid for design work. Others want to place a grid over a photo so they can draw from it.

Those are different jobs. Photoshop is strong when you are already editing a file. A free grid maker is faster when you only need to add rows and columns to a reference image and download the result.

How to make a grid in Photoshop

In Photoshop, open your image and go to View, then Show, then Grid. That turns on the grid overlay in the workspace.

If you want to change the grid spacing, open Preferences, then Guides, Grid & Slices. From there, you can adjust the grid line color, style, spacing, and subdivisions. Adobe's own Photoshop grid and guide documentation describes grids and guides as visual aids for positioning and editing, not as a finished drawing-grid export.

This is useful for design work. It helps you align shapes, check spacing, and place elements cleanly. If you are making a poster, a mockup, or a social graphic, Photoshop's grid can be enough.

When Photoshop is not the fastest choice

If your goal is drawing, the Photoshop route can feel heavy. You may only want a gridded copy of a photo. You may not need layers, rulers, guides, or a full editing workspace.

You also need to think about export. A workspace grid is helpful on screen, but it is not always the same as saving a clean image with visible grid lines. For a drawing reference, the final file matters. You want to open it, print it, or put it next to your sketchbook.

How to make a grid on a photo online

For drawing, the simpler path is to use an online grid maker. Upload the photo, choose the number of rows and columns, adjust the line color and opacity, then download the image.

That gives you a reference image with the grid already on it. You can use it on a laptop, a tablet, or a phone. You can print it. You can also keep a clean original and a gridded version side by side.

Photoshop grid vs online grid maker

Use Photoshop if you are already editing a design file and need a grid for alignment. It is good for layout work, spacing, and checking positions while you work.

Use a grid maker if you are preparing a drawing reference. It is better for quick photo grids, portrait practice, proportion studies, and classroom handouts.

  • Photoshop grid: useful inside Photoshop while editing.
  • Online grid maker: useful when you need a downloadable gridded image.
  • Photoshop: better for design layout and layered files.
  • Grid maker: better for drawing from a photo with rows and columns.

A quick drawing setup

Start with a clear photo. Use fewer cells for loose sketches and more cells for detailed faces, hands, buildings, or lettering. If the lines hide the photo, lower the opacity before making the grid thicker.

If you are not sure where to start, read this note on choosing a grid size for drawing. If the grid is hard to see, this guide to making a grid overlay easy to see covers color and opacity.
