## Features

- 🎨 Customize rows, columns, line color, thickness, and opacity
- 🔤 Add labeled drawing grids with cells like A1, B2, C3
- 📐 Create 1-point and 2-point perspective drawing grids
- 🔒 Process images 100% in the browser — nothing is uploaded
- 🆓 Free to use, with no signup required

## How to Use

1. Open [OnlineGridMaker.com](https://onlinegridmaker.com)
2. Upload a reference image
3. Adjust the grid settings
4. Download the result as PNG
