    What is a Grid Maker?

    A grid maker places evenly spaced guide lines over any image. Artists use the grid method to transfer proportions accurately — one section at a time.

    OnlineGridMaker lets you customize rows, columns, line style, and opacity — then download the result as PNG. Everything happens in your browser. Your images are never uploaded.

    FAQ

    What is a grid maker used for?

    It is used to add guide lines to a reference image so drawing proportions and placement are easier to measure.

    Is this free?

    Yes. The tool is free and does not require an account.

    Can I change color/thickness?

    Yes. You can change the grid color, line width, opacity, rows, and columns.

    Do I need to create an account?

    No. You can upload, edit, and download directly in the browser.

    How do I use it for drawing?

    Upload a reference image, set a grid density that fits the drawing, enable labels if helpful, and download the gridded file.

    Is my image uploaded anywhere?

    No. Your image is processed locally in your browser and is not sent to a server.

    What file formats are supported?

    JPG, PNG, WebP, and static GIF files up to 100MB are supported, up to 30 megapixels.

    Can I use this for Instagram grids?

    No. OnlineGridMaker is focused on drawing reference grids, not splitting images into social media tiles.
